Css display: inline-block

Trường hợp 1: khi muốn đặt thuộc tính width, height có tác dụng trên các thẻ có kiểu hiển thị mặc định là inline như a hay span thì chuyển thuộc tính display: inline-block

Các thẻ a và span có kiểu hiển thị là inline-block vẫn nằm trên một hàng giống thẻ inline và được dặt width height như thẻ block
