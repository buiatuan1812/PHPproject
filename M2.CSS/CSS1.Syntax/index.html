CSS Syntax(Cú pháp) and Selectors(Bộ chọn)

A CSS rule-set consists of a selector and a declaration block:

Một quy tắc cú pháp css bao gồm phần bộ chọn và khối khai báo code của nó

The selector points to the HTML element you want to style

Selector sẽ chỉ ra thành phần HTML bạn muốn trang trí

The declaration block contains one or more declarations separated by semicolons.

Một đoạn khai báo code chứa 1 hoặc nhiều thành phần khai báo được phân tách bởi dấu ;
